# CCN London

Website for the Cambridge Consulting Network London Office — a pro-bono student consultancy at London's top universities.

**Live:** [ccnlondon.vercel.app](https://ccnlondon.vercel.app)

## About

CCN London delivers professional-grade strategic consulting to startups and early-stage ventures at zero cost. Teams of 4-6 undergraduate, Masters, and PhD consultants from LSE, UCL, and other London universities run 8-week project cycles covering market research, go-to-market strategy, financial modelling, and pitch deck support.

Mentored by alumni from McKinsey, BCG, Bain, Deloitte, EY, OC&C, and Strategy&.

## Pages

- `index.html` - Homepage
- `about.html` - About CCN London
- `services.html` - Service offerings
- `clients.html` - Client portfolio
- `team.html` - Team directory
- `join.html` - Applications and recruitment
- `partnerships.html` - Partnership enquiries
- `contact.html` - Contact form
